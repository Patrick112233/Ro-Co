Welcome to Ro-Co's documentation!
===================================


The **Ro-Co** "Rosenheim Community" app is a cummunity driven web platform that connects students and exchange interdisciplinary knowledge.
We aim to offer students a safe space to ask questions about ther studies and private prject to encurage a maker mentatlitet by leveradge the enormus shared knoledge.
As a open-Source project we want to encourage students to contribute and help shape the platform to their needs, to create a central paltform that connects student.



Get Started
-----------

1. **Clone the repository:**

   .. code-block:: bash

      git clone https://github.com/Patrick112233/Ro-Co.git
      cd Ro-Co

2. **Start the application using Docker Compose:**

   .. code-block:: bash

      docker-compose up --build

   The first time you run this command, it may take a while to download and build all dependencies.

3. **Access the application:**

   Open your browser and go to `http://localhost:3000/` to use the Ro-Co platform.
